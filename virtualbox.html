The Joys of VirtualBox

I have been learning about virtual machines and the importance of practicing your skills with them. I decided to go the open source route with VirtualBox by Oracle.

I wanted to be meticulous in downloading VirtualBox with the Linux distro of my choice (Kali Linux). I decided to follow nine different methods to see if they were all pretty much consistent with one another while I followed along. I know that nine methods might be a little bit of an over-kill but I really needed consistency among the different methods to be sure that I was doing it right.

If the nine methods were all pretty much consistent with one another then I would follow it exactly. If one of the steps in the method was off I would try to figure out why that particular step was different from all the others.

And that is how I was able to successfully download VirtualBox with Kali Linux with no problem whatsoever. I also downloaded Linux Mint as well. Here are the nine methods I used to accomplish this:

  1. Hacker’s PC: Installing Kali Linux on VirtualBox
  2. Hacker’s PC: Configuring Kali Linux VirtualBox
  3. Building a Hack Lab For Free: Part 1
  4. How to Install Kali Linux on VirtualBox Part 1
  5. Kali Linux Virtualbox Pentest Lab
  6. How to install Kali Linux in virtual machine step by step
  7. Installing Kali Linux 1.0.6 in Virtualbox
  8. A detailed guide on installing Kali Linux on VirtualBox
  9. 10 Steps How to Create Kali Linux Virtual Machine in Virtual Box

Now that I got that down it's time to learn some stuff! And because I accomplished this feat guess what time it is? It's time for the Woohoo Dance!
