Fun Alerts

Today I learned about using JavaScript to create custom alerts and prompts. I wanted to go beyond the typical "Hello World" alert and do something a little bit different.

The JavaScript alert I used created a popup window that displayed the custom message I wanted to display. And the JavaScript prompt alert triggered a popup window asking the user interacting with it a question and then prompting the user to respond.

The script was very easy and fun to do. Since I created my first JavaScript script you know what that means right? It's time for the Woohoo Dance!
